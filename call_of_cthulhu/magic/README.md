◆ハスターの歌
　歌は吠えるような号泣のように聞こえる。
　呪文の使い手は呪文を試みるラウンドごとに1D4正気度ポイントと1D4マジック・ポイントを消費する。
　うまく歌うためにはD100を振りPOW+DEX×2以下を出さなくてはいけない。
　対象は呪文の使い手に見えていなければいけない。呪文の対象は狙われた対象だけである。
　呪文が効果を発揮するのは夜間だけでアルデバラン星が見えている時のみ。
　呪文に成功すると狙われた犠牲者の身体に火ぶくれができ1ラウンド毎に耐久力1D6のダメージを受ける。
　それから2ラウンド毎にAPPが1D6ポイント下がる。また4ラウンド毎に身体の中の火ぶくれがつぶれてCONが1D6ポイント下がる。
　この歌は防衛手段としても使用できる。誰かがこの呪文をかけてきたのに対してこの呪文で身を守る事が出来る。

 哈斯塔之歌
